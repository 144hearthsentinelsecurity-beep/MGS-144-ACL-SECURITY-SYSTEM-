## What Is MGS-144?

**MirrorGate Sentinel 144** is a pre-intake signal authentication framework for enterprise AI and SaaS pipelines.

Most security systems evaluate signals *after* intake — they filter downstream, log retroactively, and alert after the fact. The threat is already inside by the time the system responds.

**MGS-144 answers before intake.** Gate first. Intake second. Process third.

The design logic comes from biological defense: mucosal immunity. Your body doesn't wait for a pathogen to reach the bloodstream. It maintains a first-contact membrane — a standing gate — that interrogates signals before they cross the threshold.

MGS-144 applies that logic to enterprise data architecture.

## Quick Start

### Prerequisites

```bash
pip install fastapi uvicorn pydantic
```

### Environment Setup (Required — TVG Gate V)

All TVG calibration values must be set before instantiating the engine.

```bash
export TVG_BILATERAL_DELTA_THRESHOLD=<value>
export TVG_RECONCILIATION_FLOOR=<value>
export TVG_HARD_STOP_FLOOR=<value>
export TVG_SURFACE_WEIGHT=<value>
export TVG_SUBLINGUISTIC_WEIGHT=<value>
export TVG_COMPOSITE_AMPLIFIER=<value>
export TVG_CONVERGENCE_CONFIDENCE_MIN=<value>
export TVG_WINDOW_SIZE=<value>
export TVG_SESSION_DECAY_RATE=<value>
export TVG_CAMPAIGN_SENSITIVITY=<value>
export TVG_ENABLED=true
```

*TVG calibration values are available under NDA to qualified strategic acquirers only.*

### Run

```bash
python mgs_144_acl.py
```
