# Security Policy

## Reporting Vulnerabilities

This repository contains **patent-pending security architecture**.

To report a vulnerability or security concern:
- Email: dgodwin1776@gmail.com
- Do NOT open public GitHub issues for security matters

## Trade Secret Protection

The Veil Gate (TVG) — Gate V of the MGS-144 pipeline — is permanently designated
trade secret. Calibration values are loaded exclusively from environment variables.
No TVG internals appear in this repository.

## Responsible Disclosure

We support responsible disclosure and will respond to verified security researchers
within 72 hours.
