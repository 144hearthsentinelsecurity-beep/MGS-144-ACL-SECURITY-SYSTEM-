# MGS-144 Architecture — Public Reference

## Overview

MirrorGate Sentinel 144 is an 8-layer pre-intake signal authentication framework.
This document covers the public-safe architectural description.

**Note:** The Veil Gate (TVG) — Layer 7 — is trade secret and not described here.

## Layer Descriptions

### Layer 1: IAM — Ironclad AI Mirror
Reads compositional structure, not surface presentation.
- Unicode anomaly detection (homoglyph scanning)
- Shannon entropy deviation analysis
- Sub-linguistic pattern matching
- Payload embedding detection
- Coordinated marker scoring

Output: structural fact, not probabilistic assessment.

### Layer 2: AHL — Ambient Hostility Layer
Default Deny environment. IAM structural output calibrates ambient hostility.
- Base hostility coefficient: 0.15
- IAM composite amplifies the coefficient
- Acts as environmental multiplier for all downstream gates

### Layer 3: QGS-144 — QuadGate Sentinel 144
Four named gate agents, IAM-informed:
- **Michael** · Intent Classification
- **Gabriel** · Authority Verification
- **Raphael** · Semantic Integrity
- **Uriel** · Structural Analysis

### Layer 4: DKG — Convergence Engine
Named for David Kenny Godwin — Class A lineman, 45 years service.
- Channel A: lexical/surface analysis
- Channel B: sub-linguistic (IAM-fed)
- Bilateral convergence verification before current flows

### Layer 5: Alexander / Parallax 144
Multi-turn campaign arc detection.
- Reads the arc across turns, not just the individual prompt
- CRB retention enables campaign correlation across session resets
- Detects: CRESCENDO, SLOW_TIDE, INTERRUPTED campaign types

### Layer 6: Temporal Sentinel 144
Bilateral convergence verification with time-domain awareness.
- Rapid-fire flood attack detection
- Timing-based evasion detection

### Layer 7: TVG — The Veil Gate [TRADE SECRET]
Gate V · Crown Jewel · Bilateral reconciliation layer.

### Layer 8: Hard Stop Doctrine
Final verdict + CRB health update.
- TVG Gate V verdict is binding (highest priority)
- SPE threshold enforcement
- DKG divergence escalation
- Alexander campaign confirmation

## Signal Pressure Engine (SPE)

The SPE score aggregates gate signals into a unified pressure metric (0–100):

```
SPE = (avg_gate * 0.35 + IAM_composite * 0.30 + DKG_delta * 0.20 + (1 - DKG_confidence) * 0.15) * 100
```

Thresholds:
- WATCH: 18+
- FLAG: 38+
- BREACH: 55+
- HARD STOP: 72+

## CRB — Capacitor-Repeater Bond

The CRB is not a log. It is a retention bond.
- Holds structural memory of the session arc across resets
- Enables Alexander to correlate interrupted campaigns
- Health degrades on HARD STOP events, recovers on CLEAR
